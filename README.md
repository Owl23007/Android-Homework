## 📱 课程作业一：Time Clock 数字时钟

![App Screenshot](screenshoots/demo-1.png){:height="50%" width="50%"}

一个功能完善的数字时钟应用，实现了：
- 实时时间显示（时分秒）
- 智能背景切换（每4小时更换）
- 电子数字字体显示
- 完整日期信息（年月日+星期）

### 技术亮点
- 使用`Timer`和`Calendar`实现精准计时
- 动态资源加载管理背景图片
- 自定义字体应用
- 响应式UI设计
